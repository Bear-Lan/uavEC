# 无人机边缘计算调度与监控系统 - 简历项目描述

## 项目简介

**无人机边缘计算调度与监控系统 (UAV Edge Computing System)** 是一个分布式雾计算弹性调度测试平台，采用微服务架构，实现了地面数据源、无人机边缘信关站与云端服务器之间的智能算力卸载。

## 技术栈

- **后端**: Spring Boot 3.1.5 + Java 17
- **前端**: Vue 3 + TypeScript + Element Plus + ECharts
- **数据库**: MySQL 8.0 + Redis
- **通信**: WebSocket (STOMP协议) + RESTful API
- **分布式**: Redisson (分布式锁)

---

## 项目业绩（简历描述）

### 1. 分布式任务调度系统开发
- 设计并实现了基于优先级的任务调度队列，支持高优先级任务插队
- 实现了多种调度算法（贪心算法、加权公平队列、地理拓扑算法）
- 使用 Redisson 实现分布式锁，保证多节点环境下的资源分配原子性

### 2. 边缘计算卸载策略
- 实现了智能算力卸载决策：本地计算、边缘卸载、云端兜底三级降级机制
- 基于 Shannon 带宽模型进行传输效率预测
- 支持部分卸载（任务分割）：30%边缘计算 + 70%云端计算
- 实现了能耗感知和延迟感知两种卸载策略

### 3. 故障恢复与容错机制
- 实现了无人机电池耗尽自动返航（RTH）功能
- 设计了工作窃取（Work-Stealing）算法：空闲节点自动接管繁忙节点任务
- 节点宕机时自动任务迁移与重排
- 支持集群快照与回滚功能

### 4. 实时监控系统
- 基于 WebSocket 的实时数据推送，支持毫秒级状态更新
- 实现了全生命周期任务追踪：排队→调度→执行→完成
- 电池电量、网络带宽、CPU/内存使用率实时监控
- 完整的任务执行日志与性能指标分析

### 5. 前端可视化开发
- 实现了赛博朋克风格的实时雷达态势界面
- 支持无人机节点拖拽部署、缩放、平移
- 实时任务流可视化与性能指标图表
