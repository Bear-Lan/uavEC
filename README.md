# 无人机边缘计算调度与监控系统 (UAV Edge Computing System)

一个全方位的"赛博朋克风"仪表盘系统，模拟并展示了在地面数据源 (用户/IoT)、高空无人机边缘信关站 (Tier-2 集群) 以及远程云核心兜底服务器 (Tier-3) 之间的智能分布式算力卸载。

## 系统简介

本项目是一个**分布式雾计算 (Fog Computing) 弹性调度测试平台**，实现了无人机边缘计算场景下的智能算力卸载与任务调度。系统采用三级降级架构：边缘执行 → 部分卸载 → 云端兜底，确保在各种负载条件下都能保证任务的高可用性。

### 核心能力

- **智能卸载决策**：基于排队论、能耗模型、深度强化学习等多种算法，动态选择最优卸载路径
- **实时状态监控**：通过 WebSocket 毫秒级推送节点状态、任务进度、资源利用率
- **故障自愈机制**：工作窃取、任务迁移、快照回滚多级容错
- **弹性扩缩容**：支持动态添加/移除无人机节点，无需停机

## 🚀 核心特性

### 1. 智能算力卸载 (Intelligent Offloading)

系统支持三种卸载决策：无本地概念，统一为边缘/云端/部分卸载

| 决策 | 说明 | 适用场景 |
|------|------|----------|
| **EDGE** | 任务在无人机边缘节点执行 | 低延迟要求，节点资源充足 |
| **CLOUD** | 任务在云端服务器执行 | 节点资源不足或故障 |
| **PARTIAL** | 部分在边缘、部分在云端 | 复杂任务的分割计算 |

### 2. 多维卸载策略 (Offloading Strategies)

| 策略 | 算法原理 | 优化目标 |
|------|----------|----------|
| **LatencyOptimal** | 基于 M/M/1 排队论模型 | 最小化端到端延迟 |
| **EnergyOptimal** | 基于 DVFS 动态电压频率调节 | 最小化能耗消耗 |
| **AdaptivePartial** | 自适应分割任务比例 | 边缘/云端资源平衡 |
| **DQN** | 深度强化学习 (Deep Q-Network) | 长期累积奖励最大化 |

### 3. 节点管理 (Node Management)

- **动态扩缩容**：热插拔式添加/移除无人机节点
- **电池模拟**：实时模拟电池电量消耗与 RTH (Return-To-Home) 返航
- **快照回滚**：支持集群状态快照，用于故障恢复演练
- **分布式锁**：基于 Redisson 的分布式锁保护临界资源操作

### 4. 任务追踪 (Task Tracing)

精准追踪任务全生命周期耗时：
- **排队延迟** (Queue Delay)：任务等待调度的时间
- **传输延迟** (Transmission Delay)：数据传输到执行节点的时间
- **计算延迟** (Computation Delay)：任务实际执行时间
- **损耗** (Overhead)：系统开销与重试

### 5. 实时监控 (Real-time Monitoring)

- **WebSocket 推送**：毫秒级状态更新
- **雷达拓扑视图**：可视化任务链路与节点状态
- **ECharts 图表**：延迟、能耗、带宽多维指标展示

## 📊 卸载算法详解

### Shannon 带宽模型

边缘到云端的传输速率计算：
```
C = B * log2(1 + S/N)
```
其中 B 为带宽，S/N 为信噪比。

### M/M/1 排队论延迟模型

边缘节点延迟：
```
Latency_edge = 1 / (μ - λ)
```
其中 μ 为服务率，λ 为到达率。

### 能耗模型 (DVFS)

基于动态电压频率调节的能耗计算：
```
E = C * V^2 * f * t
```
其中 C 为开关电容率，V 为电压，f 为频率，t 为执行时间。

## 📦 本地启动指南

### 1) 启动后端调度微服务 (Spring Boot)

```bash
cd backend
# 确认 MySQL (3306) 和 Redis (6379) 已运行
# 检查 src/main/resources/application.properties 数据库配置
./mvnw clean spring-boot:run
```

后端服务将在 `http://localhost:8080` 启动，API 路由规范定义于 `/api/` 路径下。

### 2) 启动前端态势感知面板 (Vue 3 + Vite)

```bash
cd frontend
npm install
npm run dev
```

访问 **`http://localhost:5173`** 打开赛博朋克风格控制台。

## 🎮 平台操作指南

### 1. 防御中心 (OP-CENTER) 连线
- 进入主页，输入指挥官代号并完成拓扑基站在网格中的坐标定标
- 系统自动握手 WebSocket 通讯

### 2. 态势感知总览 (Dashboard Overview)
- 雷达界面显示指挥中心 (👤) 和无人机阵列 (✈️)
- 滚轮缩放，长按平移，无人机图标可拖拽定位

### 3. 倾泻计算任务 (Dispatching Tasks)
- 点击 **"🚀 发布突发负载"** 发布任务
- 轻型任务由无人机边缘吞下，大型任务自动分流至云端

### 4. 集群算力扩增 (Network Expansion)
- 在 **"边缘网格信标 (Cluster)"** 频道点击 **"+ 部署新站"**
- 新无人机毫秒级入网并加入调度队列

### 5. 审计微观时序 (Auditing Lifecycle)
- 在 **"战役事件追溯 (Trace)"** 查看任务流水
- 每一毫秒的排队、每一焦耳的耗电均被可视化
